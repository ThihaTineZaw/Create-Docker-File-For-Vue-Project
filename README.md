# Docker Image for Vue Project

A simple Vue project running using Docker and Docker Compose.

---

## 📥 Clone the Project

```bash
git clone https://github.com/ThihaTineZaw/Create-Docker-File-For-Vue-Project.git



🚀 Run the Project with Docker
docker compose up --build

🌐 Open in Browser
After the container is running, open:

http://localhost:8080
```
